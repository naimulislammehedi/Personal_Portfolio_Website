Roadmap for planning your portfolio project:
================================================
1) Define Purpose – Decide what you want to showcase (skills, projects, contact info).
2) Plan Structure – List sections: Hero, About, Projects, Skills, Contact.
3) Wireframe/Layout – Sketch rough layout (desktop & mobile).
4) Choose Style – Colors, fonts, spacing, and overall theme.
5) Gather Content – Text, images, project screenshots, links.
6) Build HTML – Structure each section semantically.
7) Style with CSS – Add design, responsiveness, and hover effects.
8) Test & Optimize – Check mobile, browser compatibility, and load speed.
9) Deploy – Host on GitHub Pages.
